## 4. 测试用例说明

### 4.1 完整正常流程

测试方法：

```text
completeQuestSubmissionJourneyFromRegistrationToReviewApproval
```

覆盖步骤：

1. 维护者通过 `/api/v1/auth/register` 注册。
2. 维护者通过 `/api/v1/auth/login` 登录并获取 JWT。
3. 冒险者通过 `/api/v1/auth/register` 注册。
4. 冒险者通过 `/api/v1/auth/login` 登录并获取 JWT。
5. 测试夹具创建管理员账号并生成管理员 JWT。
6. 维护者通过 `/api/v1/repositories/import` 导入仓库。
7. Mock Gitea Issue 后，通过 `/api/v1/repositories/{repositoryId}/sync` 同步 Issue。
8. 维护者通过 `/api/v1/quests` 发布任务。
9. 维护者通过 `/api/v1/quests/{questId}/submit` 提交管理员审核。
10. 管理员通过 `/api/v1/quests/{questId}/admin-reviews` 审核上架任务。
11. 冒险者通过 `/api/v1/quests` 浏览任务大厅。
12. 冒险者通过 `/api/v1/quests/{questId}` 查看任务详情。
13. 冒险者通过 `/api/v1/quests/{questId}/assignments` 接取任务。
14. Mock Gitea merged PR 后，通过 `/api/v1/quests/{questId}/submission-draft` 打开提交草稿并触发 PR 同步。
15. 冒险者通过 `/api/v1/submissions` 提交成果。
16. 维护者通过 `/api/v1/submissions/review-queue` 查看审核队列。
17. 维护者通过 `/api/v1/submissions/{submissionId}/reviews` 审核通过并给出评价项。

最终断言：

- Quest 状态为 `COMPLETED`。
- Submission 状态为 `APPROVED`。
- Assignment 状态为 `COMPLETED`。
- GrowthProfile 中 XP 和完成任务数已更新。
- XpTransaction 已写入 XP 流水。
- ContributionRecord 已写入贡献记录。
- ReviewRecord 已写入审核评价记录。

### 4.2 异常流程：未登录访问

测试方法：

```text
protectedWorkflowEndpointsShouldRejectAnonymousAccess
```

覆盖接口：

- 未登录接单：`POST /api/v1/quests/{questId}/assignments`
- 未登录提交成果：`POST /api/v1/submissions`
- 未登录访问审核队列：`GET /api/v1/submissions/review-queue`

断言结果：

- HTTP 状态码为 `401 Unauthorized`。
- 响应错误码为 `UNAUTHORIZED`。

### 4.3 异常流程：重复接单

测试方法：

```text
duplicateAssignmentShouldBeRejected
```

覆盖步骤：

1. 构造并上架一个可接取任务。
2. 冒险者第一次接单成功。
3. 同一冒险者第二次接单。

断言结果：

- 第二次接单返回 `409 Conflict`。
- 响应错误码为 `DUPLICATE_ASSIGNMENT`。
- 数据库中该用户对该任务只有一条 `ACTIVE` 接取记录。

### 4.4 异常流程：PR 未合并禁止审核通过

测试方法：

```text
approvalShouldBeRejectedWhenPullRequestIsNotMerged
```

覆盖步骤：

1. 构造并上架一个任务。
2. 冒险者接单。
3. Mock Gitea PR 为 `OPEN` 且 `merged=false`。
4. 冒险者打开提交草稿并提交成果。
5. 维护者尝试直接审核通过。

断言结果：

- 审核通过请求返回 `409 Conflict`。
- 响应错误码为 `PR_NOT_MERGED`。
- Quest 保持 `IN_REVIEW`。
- Submission 保持 `PENDING_REVIEW`。
- 未生成 GrowthProfile。
- 未生成 ContributionRecord。
