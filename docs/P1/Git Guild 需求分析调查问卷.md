# Git Guild 需求分析调查问卷

您好！我们正在开展《软件工程与计算 II》课程项目 Git Guild 的需求分析工作。Git Guild 是一个面向开源新手的游戏化任务协作代码托管平台。在 GitHub 等代码托管平台的功能基础上，我们添加了"冒险家公会"的概念，提供任务发布和接取等功能，目标是帮助用户通过完成真实任务逐步积累经验、提升技能，并获得成长反馈。本问卷采用匿名方式，仅用于课程项目的需求调研与分析，不涉及任何商业用途。感谢您的参与和支持。
