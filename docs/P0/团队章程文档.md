## AI 工具链选型及理由
- AI 编程助手：选择 GitHub Copilot、Codex（后端）和 Claude Code（前端）。理由是 GitHub Copilot 有学生免费计划并且可使用多种模型，而 Codex 在后端开发方面表现优异，Claude Code 在前端开发方面表现出色。
- AI 对话工具：选择 ChatGPT、Claude 和豆包，用于文档生成、需求分析。理由是豆包在中文文本处理方面表现优异，ChatGPT 和 Claude 在多语言支持和上下文理解方面表现出色。
- AI 代码审查工具：选择 CodeRabbit，因为可以自动化代码审查流程，提供详细的审查报告，综合来看是最强的代码审查工具。
- AI 测试工具：选择 Apifox AI，因为它支持后端 API 全量测试用例生成，自动生成接口自动化脚本，并且支持 Git 仓库、PR、代码审查等业务场景用例，贴合我们项目选题。
