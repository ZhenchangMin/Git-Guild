## 3. AI 工具链选型

### 3.1 AI 编程助手

- 选用工具：GitHub Copilot、Codex、Claude Code
- 选择理由：
  - GitHub Copilot 有学生免费计划，且可使用多种模型。
  - Codex 在后端开发方面表现优异。
  - Claude Code 在前端开发方面表现出色。

### 3.2 AI 对话工具

- 选用工具：ChatGPT、Claude、豆包
- 使用场景：文档生成、需求分析
- 选择理由：
  - 豆包在中文文本处理方面表现优异。
  - ChatGPT 和 Claude 在多语言支持与上下文理解方面表现出色。

### 3.3 AI 代码审查工具

- 选用工具：CodeRabbit
- 选择理由：支持自动化代码审查流程，能够提供详细的审查报告。

### 3.4 AI 测试工具

- 选用工具：Apifox AI
- 选择理由：
  - 支持后端 API 全量测试用例生成。
  - 支持自动生成接口自动化脚本。
  - 支持 Git 仓库、PR、代码审查等业务场景用例，贴合项目选题。
